# Project Summary

This repository contains a lightweight, academic-grade pipeline for monitoring and auditing data access in a simulated big-data environment. It combines a structured access-log dataset, a Jupyter notebook for exploratory analysis, a Python anomaly detector, and supporting report artifacts.

## What is included
- Simulated access log dataset
- Exploratory analysis notebook
- Rule-based anomaly detection script
- Prebuilt Excel dashboard and supporting reports
- Regression tests for the anomaly detection workflow

## Intended use
The project is suitable for demonstrating concepts in security monitoring, audit trails, SIEM-style alerting, and anomaly detection in a reproducible and accessible format.
